# 8장. 생성 모델을 데이터 분석에 연결하기

**🎯 학습 목표: GAN, VAE, Diffusion Model의 기본 아이디어를 이해하고, 합성 데이터 생성과 데이터 증강, 이상 탐지, 불확실성 분석에 어떻게 연결되는지 설명할 수 있다**

---

## 🌟 이 장에서 배울 내용 미리보기

- 생성 모델이 왜 분석 도구가 될 수 있는지
- CTGAN이 정형 데이터 합성에 쓰이는 방식
- VAE가 압축과 생성, 이상 탐지를 함께 다루는 방식
- Diffusion Model이 예측과 불확실성 정량화에 연결되는 방식

---

## 8.1 생성 모델은 "새 데이터를 만들어 보는 분석 도구"다

생성 모델은 단순히 그림을 그리는 AI가 아니다. 데이터의 구조를 학습한 뒤, 그 구조와 비슷한 새 표본을 만들 수 있다는 점이 핵심이다.

이 능력은 데이터 분석에서 다음 같은 문제에 유용하다.

- 데이터가 적을 때 샘플을 보강
- 민감 정보를 직접 공개하지 않고 유사한 데이터 생성
- 희귀 사례를 늘려 학습 균형 개선
- 미래 시나리오나 불확실성 범위를 탐색

즉, 생성 모델은 "창작 도구"이면서 동시에 "실험용 시뮬레이터"다.

---

## 8.2 GAN은 경쟁을 통해 더 그럴듯한 데이터를 만든다

GAN은 생성자와 판별자가 경쟁하며 학습한다.

- 생성자: 가짜 데이터를 만든다.
- 판별자: 진짜와 가짜를 구분한다.

비유하면 위조지폐 제작자와 감식관이 서로 수준을 높여 가는 과정과 비슷하다. 이 경쟁이 잘 작동하면 점점 더 실제와 비슷한 데이터가 생성된다.

정형 데이터에서는 일반 GAN보다 CTGAN 같은 변형이 더 자주 쓰인다. 범주형 변수와 복잡한 분포를 더 잘 다루기 때문이다.

---

## 8.3 VAE는 압축과 생성을 동시에 한다

VAE는 입력 데이터를 잠재 공간에 압축한 뒤 다시 복원한다. 이 과정에서 잠재 공간이 잘 정리되면, 그 공간에서 샘플을 뽑아 새 데이터를 생성할 수 있다.

비유하면, 복잡한 사물을 몇 개의 핵심 좌표로 요약해 두고 그 좌표를 조금 움직여 비슷한 새 예시를 만드는 것과 같다.

그래서 VAE는 다음과 잘 연결된다.

- 데이터 증강
- 잠재공간 시각화
- 재구성 오차 기반 이상 탐지

---

## 8.4 Diffusion Model은 노이즈 제거를 거꾸로 학습하는 방식이다

Diffusion Model은 데이터를 조금씩 망가뜨리는 과정을 가정하고, 그 반대로 원래 데이터를 복원하는 법을 학습한다.

쉽게 말하면, 흐릿해진 사진을 단계적으로 선명하게 되돌리는 방법을 배우는 것이다.

이 방식은 생성뿐 아니라 불확실성 정량화에도 유리하다. 여러 번 샘플을 생성해 보면 "어디까지가 안정적인 예측이고 어디부터가 불확실한가"를 파악할 수 있다.

```text
1. 생성 모델은 단순 콘텐츠 생성기가 아니라 데이터 분석용 시뮬레이션 도구가 될 수 있다.
2. GAN은 경쟁을 통해, VAE는 잠재공간 학습을 통해, Diffusion은 노이즈 제거를 통해 데이터를 생성한다.
3. 생성 모델은 합성 데이터 생성, 데이터 증강, 이상 탐지, 불확실성 분석에 활용될 수 있다.
```
