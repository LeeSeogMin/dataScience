# Lecture Directory

`lecture/`는 최종 강의자료 원고를 두는 폴더다. 현재 단계에서는 본문 집필 전에 필요한 기준 문서만 둔다.

## 구성

- `README.md`: 폴더 목적과 운영 원칙
- `STYLE_GUIDE.md`: `causal/lecture` 스타일을 따르는 집필 규칙
- `TEMPLATE.md`: 장별 원고 작성 템플릿

## 작성 원칙

- `docs/`는 참고용 원고다.
- `lecture/`에는 재구성된 강의자료만 둔다.
- 각 장 파일명은 `chapter01.md` 형식으로 맞춘다.
- 문서 작성은 `STYLE_GUIDE.md`와 `TEMPLATE.md`를 먼저 확인한 뒤 시작한다.
