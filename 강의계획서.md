## 수업 목표

Python 기반 데이터 분석 환경에서 전통적 통계/머신러닝 기법과 최신 AI 도구를 함께 이해하고, 실제 데이터 문제에 맞는 분석 방법을 선택하고 검증할 수 있다.

1. 데이터 사이언스의 전체 흐름과 주요 방법론의 역할을 설명할 수 있다
2. 데이터 전처리, 특성 공학, 분류, 군집화, 앙상블, 토픽 모델링, 딥러닝, 생성 모델, 시계열, 생존 분석, 그래프 분석을 실습과 함께 이해할 수 있다
3. Python 실습 코드를 읽고 실행하며 결과를 해석할 수 있다
4. LLM과 같은 AI 도구를 분석 보조 도구로 활용하되, 결과를 검증하고 한계를 설명할 수 있다
5. 문제에 따라 전통적 방법과 AI 기반 방법 중 적절한 접근을 선택할 수 있다

## 수업 방식

> **"개념을 배우고, 바로 실습으로 확인하고, 결과를 해석한다."**

- 매 수업은 **이론 설명 + 실습 + 결과 해석**의 흐름으로 진행한다
- 이론과 실습은 분리하지 않고, 관련 개념 직후에 실습을 연결한다
- 강의자료(`lecture/`)와 실습 코드(`practice/`)를 함께 사용한다
- 실습은 단순 실행이 아니라 **왜 이런 결과가 나왔는지 해석하는 것**까지 포함한다

### AI 도구 사용 원칙

- AI 도구 사용은 허용하며, 필요 시 적극 활용할 수 있다
- 단, AI의 출력은 반드시 검토하고 검증해야 한다
- LLM 기반 분석 결과는 가설 초안으로 다루고, 최종 판단은 데이터와 코드 실행 결과로 확인한다
- 민감 데이터나 보안 이슈가 있는 경우 외부 API 사용에 주의한다
